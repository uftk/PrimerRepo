LABORATORIO DOCKER Y TRIVY CON PIPELINE EN GITHUB ACTIONS



Este proyecto implementa una aplicación web en .NET que responde con un mensaje Hola Mundo. La aplicación es empaquetada mediante Docker y se integra a un pipeline de seguridad usando GitHub Actions y Trivy.



1. Creación del dockerfile para aplicación web mínima en .NET Mensaje Hola Mundo. Se eligio Python 3.8
2. Configuración de un worflow de GitHub actions.
3. Creación de parseo para leer json de Trivy e imprimir reporte en log de pipeline.
4. Descripción en readme
